建網站

首先先到mde.tw 網站第一頁，點選下面連結

 

點選此檔案，裡面老師有詳細的教學內容

 

 

首先先下載隨身系統

fall2020(老師有做好的路徑及pip指令)

 

或是直接至官網下載

載完後解壓縮會有這四個檔案

確認python版本

在Y槽打上python

確認版本後用ctrl+Z跳出

 

再確認擁有的pip附加檔案

 

再到git 官網下載portable git

將其下載下來

並且解壓縮到data (自行新增一個新的資料夾)

 

 

將開始的路徑做更動(將其拉的小白)

打上

此處路徑要一樣

宣告變數函數path_git的路徑為%Disk%:\portablegit\bin;

 

再路徑下多打上%path_git%;，才能拿來當作路徑使用

 

打完後做儲存，並重開

 

再來設定git config (有對應的語法必須查Google找git windows
